# Daily Planner

Aplikasi To-Do List sederhana berbasis Java untuk mengelola tugas harian melalui terminal.

## Fitur Utama
- [x] **Tambah Tugas**: Input judul tugas baru (dengan validasi nama tidak boleh kosong).
- [x] **Lihat Daftar**: Menampilkan semua tugas beserta statusnya.
- [x] **Tandai Selesai**: Mengubah status tugas menjadi selesai dengan simbol `[V]`.
- [x] **Hapus Tugas**: Menghapus tugas tertentu dari daftar.
- [x] **Anti-Crash**: Dilengkapi dengan Error Handling (Try-Catch) untuk menangani input non-angka.

## Cara Menjalankan
1. Clone repositori ini.
2. Compile file menggunakan `javac Main.java`.
3. Jalankan aplikasi dengan `java Main`.
